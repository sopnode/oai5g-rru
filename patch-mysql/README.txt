This directory contains the mysql database information for OAI5G from
https://gitlab.eurecom.fr/oai/cn5g/oai-cn5g-fed.git

- oai_db-basic-orig.sql corresponds to the original file from
https://gitlab.eurecom.fr/oai/cn5g/oai-cn5g-fed/-/blob/master/charts/oai-5g-core/mysql/initialization/oai_db-basic.sql

- oai_db-basic-generic.sql is the corresponding generic version
that will be patched by the configure-demo-oai.sh script to map
the input parameters of the scenario.
